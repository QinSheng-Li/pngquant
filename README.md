# pngquant
a pngquant plug for cocos creator

support all(web-mobile,web-desktop,jsb-default,jsb-binary,jsb-link) build project!

## Usage:
1. build cocos creator project!
2. open plug panel!
3. choose platform you want!
4. click compression button!
5. enjoy!

## Issue:
